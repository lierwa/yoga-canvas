# @yoga-canvas/react

## 0.1.1

### Patch Changes

- **React 绑定层初始化**:
  - 提供 `YogaCanvas` React 组件，实现 Canvas 渲染的 React 式声明写法。
  - 内置 `useYogaCanvas` Hook，支持在 React 中操作 Canvas 实例。
  - 完善的编辑器辅助功能：节点选择高亮 (Selection Overlay)、旋转与缩放手柄、无限画布等。
  - 支持 Tailwind CSS 到 Yoga 样式的转换工具，提升样式开发体验。
  - 提供 JSX 到 Canvas Node Tree 的双向转换逻辑。
- Updated dependencies
  - @yoga-canvas/core@0.1.1
