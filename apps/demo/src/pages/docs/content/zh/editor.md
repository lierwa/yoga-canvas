# Demo / 编辑器

- Yoga Canvas 演示项目不仅是一个展示平台，更是一个功能完备的**可视化编辑器原型**。你可以直接在此创建、编辑并导出你的 Canvas 布局。
![编辑页面](/src/pages/docs/assets/editor.png)
---

## 1. 项目管理 (WorkSpace)

在首页或侧边栏，你可以对项目进行全面管理：
- **CRUD 操作**: 支持创建、复制、重命名和删除项目。
- **持久化**: 所有的编辑内容都自动保存在浏览器的 `localStorage` 中，无需担心进度丢失。
- **模板示例**: 内置了多个预设场景，方便你快速了解 Flex 布局在 Canvas 中的表现。
---

## 2. 核心编辑功能

编辑器由三个核心面板组成：

### 节点树 (Node Tree)
- **可视化层级**: 像 Chrome 开发者工具一样查看所有节点的嵌套关系。
- **快速选中**: 点击树节点即可在画布上精准定位。

### 属性面板 (Property Panel)
- **Flex 布局调参**: 实时修改 `flex-direction`、`justify-content`、`gap` 等布局属性。
- **视觉样式配置**: 调整背景色、圆角、渐变、阴影等视觉参数。
- **Tailwind 支持**: 直接在 `className` 输入框中键入 Tailwind 类名，效果即时呈现。

### 画布交互 (Interactive Canvas)
- **高亮反馈**: 鼠标悬浮或选中节点时，画布会显示精确的边框和尺寸标注。
- **视口缩放**: 支持通过手势或快捷键对画布进行缩放和平移。

---

## 3. 预览与导出

- **多端预览**: 实时查看你的布局在 H5 甚至模拟的小程序环境下的表现。
- **代码生成**: 自动生成可直接复制的 JSX、JSON 或标准的 HTML/CSS 代码。
- **一键下载**: 将编辑成果导出为高清图片。

---

## 4. 开发者参考

如果你计划开发自己的可视化设计工具（如海报编辑器、大屏编辑器），本 Demo 的源码将是极佳的参考：
- **状态同步**: 学习如何使用 React 状态管理复杂的节点树变更。
- **命令模式**: 参考撤销/重做功能的潜在实现思路。
- **性能优化**: 了解如何处理高频交互下的画布重绘。
