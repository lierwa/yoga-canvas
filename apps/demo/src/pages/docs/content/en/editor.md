# Demo / Editor

The demo includes:

- Workspace: project management (create/duplicate/rename/delete), persisted in localStorage
- Editor: node tree, selection highlight, zoom/pan, property tweaking
- Preview/Export: preview canvas and export JSON/HTML/TS

Treat it as:

- A regression playground for layout/render/interaction/export
- A reference implementation for your own editor/design tool
